# grpc-web-html
## Start Client
`$ phyton -m simpleHTTPServer 8081`  
- loacalhost:8081 をブラウザーで起動その後clientのディレクトリーに行くとサンプルが起動します。
## Start Server
`$ docker-compose up`  
- 上記のコマンドでサーバー側を起動

`$ docker-compose down`  
- 上記のコマンドで立ち上げてるサーバーを停止させる
## Build Client
`$ cd client`  
`$ npm install`
## Build Server
`$ cd server`  
`$ npm install`
